# 🤫 QuietZone - نظام ذكي للحد من التلوث الضوضائي

نظام متكامل يعتمد على الذكاء الاصطناعي وإنترنت الأشياء (IoT) لمراقبة مستويات الضجيج في الأماكن العامة المغلقة، وتحليلها، وتنبيه المستخدمين بضرورة الهدوء للحفاظ على بيئة مريحة.

## 🚀 عن المشروع
مشروع **QuietZone** يهدف إلى حل مشكلة التلوث الضوضائي في المكتبات، المستشفيات، ومساحات العمل المشتركة. يقوم التطبيق بقياس مستوى الديسيبل (dB) في الوقت الفعلي، وتصنيف نوع الضجيج، وتوفير خريطة حرارية (Heatmap) للمناطق الهادئة.

## ✨ الميزات الرئيسية
- **تحليل حي للصوت:** قياس مستويات الضجيج بدقة باستخدام ميكروفون الهاتف.
- **تصنيف الضوضاء:** تمييز أنواع الصوت (كلام بشري، موسيقى، صراخ، أو هدوء) عبر خوارزمية تحليل الطيف.
- **تنبيهات لحظية:** إرسال إشعارات Push Notifications فور تخطي الضجيج للحد المسموح.
- **العصا السحرية (Magic Wand):** توجيه المستخدم تلقائياً إلى أقرب منطقة هادئة مرصودة في محيطه.
- **الخريطة الحرارية:** عرض بيانات الضجيج التاريخية على الخريطة لتسهيل اختيار المكان المناسب.

## 🛠 التقنيات المستخدمة
- **Backend:** Node.js, Express.js
- **Real-time Communication:** Socket.io
- **Database:** MongoDB Atlas
- **Mobile App:** React Native, Expo
- **AI/DSP:** خوارزمية تحليل الطيف الترددي (Audio Spectrum Analysis)
- **Maps:** Google Maps API & OpenStreetMap
